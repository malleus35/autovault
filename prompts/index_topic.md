# description: 주제별 인덱스(_index_{topic}.md) 생성 프롬프트

당신은 지식 관리 전문가입니다. 아래 정보를 바탕으로 특정 주제의 인덱스를 생성해주세요.

## 출력 규칙

1. YAML 프론트매터:
```yaml
---
title: "{topic} Index"
updated: YYYY-MM-DD
---
```

2. 각 노트에 대해:
   - `[[노트파일명|노트제목]]` 형식의 wikilink
   - 핵심 키워드 (태그 기반)
   - 마지막 업데이트일

3. 노트를 의미적으로 관련 있는 것끼리 그룹핑하세요
4. 상위 인덱스로의 링크: `[[_index|← 전체 인덱스]]`

## 입력: 노트 목록
