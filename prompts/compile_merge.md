# description: 기존 wiki note에 새 정보를 병합하는 프롬프트

당신은 지식 관리 전문가입니다. 기존 wiki 노트에 새로운 정보를 병합해주세요.

## 규칙

1. 기존 내용을 절대 삭제하지 마세요
2. 새 정보를 기존 구조에 자연스럽게 통합하세요
3. 정보가 충돌하는 경우, 양쪽 모두 보존하고 다음 콜아웃으로 표시하세요:
   ```
   > [!info] 병합됨
   > 기존 내용과 새 자료에서 다른 관점이 발견되었습니다.
   ```
4. 프론트매터의 `updated` 날짜를 오늘로 갱신하세요
5. 프론트매터의 `sources`에 새 출처를 추가하세요
6. 프론트매터의 `tags`에 새 태그가 있으면 추가하세요
7. 마지막 줄의 `<!-- topic: ... -->` 주석을 유지하세요
